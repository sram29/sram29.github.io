Projects: Projects of Interest

The purpose of this of this project is to compare latitude versus different weather conditions (i.e. temperature, cloudliness, humidity, and wind speed).

Some of the packages and API used are citipy python package to gather a sample of city names and their geographic coordinates, as well as OpenWeatherMap API, which contains realtime weather data.

The data was then converted to a dataframe using Pandas from which plots were created.
